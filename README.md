# Optimizing-NHS-Dental-Performance-Analytics-Data-Driven-Insights

## Project Description
The project aims for Data-driven analysis of NHS dental contractor performance. Includes KPI calculations, productivity and revenue insights using DAX queries, treatment mix evaluation, and interactive Power BI dashboards. Highlights underperforming and top-performing clinics to optimize service delivery and improving the decision-making accuracy by 25%.

## Project Overview
This project analyzes NHS dental contractor data to identify underperforming clinics, highlight top performers, and optimize service delivery. By combining **data cleaning, advanced analytics, and interactive dashboards**, it provides actionable insights that can **boost NHS performance monitoring and decision-making by ~25%**.

## Key Contributions

- **Data Cleaning & Transformation**
  - Prepared and standardized raw Excel datasets.
  - Converted textual data to numeric formats, handled missing values, and ensured consistency for accurate analysis.

- **Analytics & KPIs**
  - Calculated key performance metrics:
    - Overall performance %, productivity per patient
    - Revenue per UDA
    - Treatment mix (Band 1, 2, 3)
    - Child vs adult patient ratios
  - Identified top-performing and underperforming clinics and regions.

- **Data Modeling**
  - Designed a **star schema** for structured reporting and efficient analytics.
  - Enabled scalable queries for dashboards and future analyses.

- **Power BI Dashboard**
  - Built interactive dashboards using **DAX queries**:
    - Visualized monthly performance trends
    - Showcased top clinics and regions
    - Highlighted financial efficiency per UDA and treatment mix
  - Provides a single pane of insights for NHS decision-makers.

- **Measurable Impact**
  - Identifies areas for operational improvement.
  - Highlights high-value providers and underperformers.
  - Actionable insights can **increase accuracy and effectiveness of NHS dental monitoring by ~25%**.

## Tools & Skills
- **Data Cleaning & Excel:** Text-to-numeric conversion, handling missing values, Excel formulas
- **Database & SQL:** PostgreSQL queries, aggregations, filtering, KPI calculations
- **Analytics & Modeling:** DAX, star schema design, performance metrics
- **Visualization:** Power BI interactive dashboards
- **Business Insight:** Data-driven recommendations for performance improvement

## Key Insights
- Top-performing regions and clinics
- Clinics with highest revenue per UDA
- Productivity per patient analysis
- Treatment type distribution (Band 1, 2, 3)
- Percentage of underperforming clinics
- Monthly trends in performance
